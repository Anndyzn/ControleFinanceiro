Controle Financeiro

Projeto desenvolvido para prática de desenvolvimento Full Stack utilizando:

HTML
CSS
JavaScript
TypeScript
Node.js
Express
SQLite
Objetivo

O objetivo do projeto é criar um sistema de controle financeiro simples, permitindo registrar movimentações financeiras e visualizar informações organizadas por mês e categoria.

Funcionalidades
Cadastro de receitas, despesas e investimentos
Listagem de lançamentos
Resumo financeiro:
Saldo
Receitas
Despesas
Investimentos
Filtro por ano e mês
Interface responsiva para dispositivos móveis
Integração entre frontend e backend
Persistência de dados utilizando SQLite
Tecnologias utilizadas
Frontend
HTML
CSS
JavaScript
Backend
Node.js
Express
TypeScript
Banco de Dados
SQLite
